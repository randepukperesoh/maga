## МКЭ Решатель - Метод конечных элементов для стержневых систем (2D)

### Общая концепция

Реализация метода конечных элементов на бэкенде для расчета:

- Перемещений узлов
- Внутренних усилий в стержнях
- Напряжений
- Коэффициентов запаса прочности

**Почему на бэкенде:**

- Точность вычислений с плавающей точкой
- Возможность решения больших систем (до 1000+ элементов)
- Централизованное обновление алгоритмов
- Интеграция с нейросетью
