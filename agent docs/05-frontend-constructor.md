# Основной фронтенд - Конструктор стержней

## Технологии

- React 18
- TypeScript
- Zustand
- Fabric.js (2D canvas)
- TailwindCSS
- shadcn/ui (обязательно для базовых UI-компонентов)
- Recharts
- @react-pdf/renderer

## UI/UX требования

- Для интерфейсов использовать `shadcn/ui` как основной источник базовых компонентов.
- Визуальный стиль должен ориентироваться на подход студии Chulakov: аккуратная типографика, чистая иерархия, выразительные акценты, без перегруженного декора.

## Компоненты

### RodCanvas.tsx - основной канвас

```typescript
// Основные функции:
- Инициализация Fabric.js канваса
- Обработка кликов для добавления узлов
- Обработка выделения элементов
- Отрисовка сетки
- Масштабирование (Ctrl + колесо)
- Панорамирование (пробел + перетаскивание)
- Snap to grid
```
