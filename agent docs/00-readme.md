## Как использовать этот промпт

1. ИИ-агент должен прочитать все файлы последовательно
2. Создать структуру папок согласно [04-folder-structure.md](04-folder-structure.md)
3. Реализовать функциональность согласно соответствующим файлам
4. Использовать технологии из [03-technology-stack.md](03-technology-stack.md)
5. Запустить через Docker согласно [12-docker-deployment.md](12-docker-deployment.md)
