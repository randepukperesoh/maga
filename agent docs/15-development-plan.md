# 15-development-plan.md

## План разработки и приоритеты

### Общая концепция

Поэтапный план разработки с разбивкой на спринты. Каждый спринт — законченная функциональность, готовая к тестированию.

---

### Спринт 1: Основа проекта (1-2 дня)

**Цель:** Создание базовой инфраструктуры монорепозитория

| Задача                              | Описание                             | Приоритет |
| ----------------------------------- | ------------------------------------ | --------- |
| Инициализация монорепозитория       | pnpm workspace, структура папок      | Critical  |
| Настройка Docker Compose            | Контейнеры: postgres, redis, backend | Critical  |
| Базовый FastAPI сервер              | healthcheck, CORS, логирование       | Critical  |
| Базовый React приложение (web)      | Vite, Tailwind, роутинг              | Critical  |
| Базовый React приложение (training) | Vite, Tailwind                       | High      |
| Настройка shared-types              | Общие TypeScript интерфейсы          | High      |
| Настройка Git и CI                  | .gitignore, pre-commit хуки          | Medium    |

---

| Задача                         | Описание                                    | Приоритет |

| Визуализация результатов       | Цветовая карта, эпюры (Recharts)            | High      |
| Деформированная схема          | Наложение на исходную                       | High      |
| Таблица результатов            | Отображение цифр                            | High      |

### Приоритеты задач

| Приоритет    | Описание                      | Действие при срыве сроков     |
| ------------ | ----------------------------- | ----------------------------- |
| **Critical** | Без этого система не работает | Должны быть выполнены         |
| **High**     | Основная функциональность     | Перенести на следующий спринт |
| **Medium**   | Улучшения UX                  | Можно отложить                |
| **Low**      | Дополнительные фичи           | В следующей версии            |

---

### Риски и их митигация

| Риск                                          | Вероятность | Влияние | Митигация                                              |
| --------------------------------------------- | ----------- | ------- | ------------------------------------------------------ |
| Дип.ipynb несовместим с текущими библиотеками | Средняя     | Высокое | Использовать виртуальное окружение, фиксировать версии |
| Производительность МКЭ при >100 элементах     | Средняя     | Среднее | Оптимизация матриц, разреженное хранение               |
| WebSocket для обучения нестабилен             | Низкая      | Среднее | Fallback на polling                                    |
| Fabric.js конфликты при перетаскивании        | Средняя     | Среднее | Тщательное тестирование, изоляция обработчиков         |
| Docker объем >2GB                             | Средняя     | Низкое  | Многослойная сборка, alpine образы                     |
