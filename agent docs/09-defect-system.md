# 09-defect-system.md

## Система дефектов

### Общая концепция

Система позволяет:

- Задавать различные типы дефектов для стержней
- Моделировать влияние дефектов на напряженно-деформированное состояние
- Визуализировать дефекты на схеме конструкции
- Рассчитывать коэффициенты концентрации напряжений

---

### Типы дефектов

#### 1. Трещина

**Параметры:**

- `position`: относительная координата от 0 до 1 (начало стержня)
- `depth`: глубина трещины (мм)
- `angle`: угол наклона трещины (градусы)

**Влияние на расчет:**

- Коэффициент концентрации напряжений Kt = f(depth/width)
- Локальное увеличение напряжений: σ_max = Kt \* σ_nominal

#### 2. Коррозия

**Параметры:**

- `position_start`: начало зоны коррозии (0-1)
- `position_end`: конец зоны коррозии (0-1)
- `depth`: глубина коррозии (мм)
- `width`: ширина зоны (мм)

**Влияние на расчет:**

- Локальное уменьшение площади сечения
- Неравномерное распределение напряжений

#### 3. Уменьшение сечения

**Параметры:**

- `position`: координата (0-1)
- `new_area`: новая площадь сечения (м²)
- `length`: длина участка с измененным сечением (мм)

**Влияние на расчет:**

- Пересчет напряжений на участке: σ = N / A_new

#### 4. Неоднородность материала

**Параметры:**

- `position_start`: начало зоны (0-1)
- `position_end`: конец зоны (0-1)
- `E_reduced`: измененный модуль упругости (Па)

**Влияние на расчет:**

- Локальное изменение жесткости
- Перераспределение усилий

### API эндпоинты для дефектов

| Метод  | Эндпоинт                     | Описание                   |
| ------ | ---------------------------- | -------------------------- |
| POST   | `/api/v1/defect/add`         | Добавить дефект            |
| PUT    | `/api/v1/defect/{id}`        | Обновить дефект            |
| DELETE | `/api/v1/defect/{id}`        | Удалить дефект             |
| GET    | `/api/v1/defect/{rod_id}`    | Получить дефекты стержня   |
| POST   | `/api/v1/defect/recalculate` | Пересчет с учетом дефектов |
