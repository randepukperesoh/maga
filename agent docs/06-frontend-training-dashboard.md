# 06-frontend-training-dashboard.md

## Фронтенд дашборда обучения нейросети

### Общая концепция

Отдельный веб-интерфейс для управления обучением нейросети, который позволяет:

- Просматривать и редактировать датасет для обучения
- Запускать и останавливать обучение модели
- Мониторить процесс обучения в реальном времени
- Управлять версиями обученных моделей

### UI/UX требования

- Для интерфейсов использовать `shadcn/ui` как основной источник базовых компонентов.
- Визуальный стиль дашборда должен ориентироваться на подход студии Chulakov: аккуратная типографика, чистая иерархия, выразительные акценты.
