# 16-implementation-log.md

## Что уже реализовано

Дата обновления: 2026-04-01

### Монорепозиторий и инфраструктура

- Создан workspace (`pnpm-workspace.yaml`, корневой `package.json`)
- Добавлен `docker-compose.yml` с сервисами: `postgres`, `redis`, `backend`, `web`, `training-dashboard`, `nginx`
- Добавлен nginx reverse-proxy (`infra/nginx/default.conf`)

### Основной фронтенд (apps/web)

- Инициализировано Vite + React + TypeScript приложение
- Реализован MVP канвас на Fabric.js (`RodCanvas.tsx`):
  - отображение сетки
  - добавление узлов по клику
  - snap-to-grid
  - выбор узлов для создания стержня
  - отрисовка стержней и узлов по состоянию store
- Добавлен Zustand store (`editorStore.ts`) для узлов/стержней
- Добавлен запуск расчета из UI (`App.tsx`) через `POST /api/v1/calculate`
- Добавлено управление дефектом в UI:
  - выбор типа дефекта
  - задание глубины
  - задание позиции `position` (0..1) вдоль стержня
  - добавление дефекта к первому стержню
  - синхронизация дефектов с backend
  - удаление дефектов из UI и backend
  - редактирование дефекта через `PUT /api/v1/defect/{id}`
  - запуск `POST /api/v1/defect/recalculate`
  - вывод обновленных результатов
- Добавлена визуализация дефектов и опасности:
  - цветовая индикация стержней по |напряжению|
  - красные маркеры дефектов на стержнях
  - маркер ставится в фактической позиции по `position`
- Добавлена генерация и скачивание PDF отчета кнопкой `Скачать PDF`
- Добавлен AI-блок (моковые/notebook-informed данные):
  - кнопка `AI-прогноз`
  - панель top-рисков
  - отображение риска цветом на канвасе
- Добавлены механизмы сохранения/восстановления проекта:
  - LocalStorage: `Сохранить проект`, `Загрузить проект`, `Сбросить`
  - JSON file: `Экспорт JSON`, `Импорт JSON`
  - сохраняются узлы, стержни, дефекты, напряжения, AI-risk
- Выполнена оптимизация frontend bundle (`manualChunks`)

### Фронтенд дашборда обучения (apps/training-dashboard)

- Подключен к backend training API
- Добавлена форма запуска обучения
- Добавлен auto refresh статуса/истории
