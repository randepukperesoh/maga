## Интеграция нейросети из Дип.ipynb

### Общая концепция

Интеграция существующей нейросети из файла `Дип.ipynb` в бэкенд FastAPI для:

- Предсказания местоположения дефектов по данным о конструкции
- Визуализации вероятности дефекта в виде тепловой карты
- Сравнения предсказаний с реальными дефектами
- Возможности дообучения модели на новых данных
