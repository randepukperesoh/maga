# 11-pdf-report.md

## Генерация PDF отчетов

### Общая концепция

Система генерации профессиональных инженерных отчетов в формате PDF, содержащих:

- Схему конструкции
- Результаты расчетов (таблицы и графики)
- Информацию о дефектах
- Результаты нейросети
- Заключение

---

### Технологии

| Компонент | Технология          | Назначение                          |
| --------- | ------------------- | ----------------------------------- |
| Фронтенд  | @react-pdf/renderer | Генерация PDF на клиенте            |
| Бэкенд    | ReportLab           | Альтернативная генерация на сервере |
| Экспорт   | html2canvas + jspdf | Экспорт канваса в PNG               |
