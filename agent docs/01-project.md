# Обзор проекта

## Название

Rod System Designer — конструктор стержневых систем с МКЭ расчетами и нейросетевой детекцией дефектов

## Цель

Создать веб-приложение для инженеров-механиков и строителей, позволяющее:

- Визуально конструировать 2D стержневые системы
- Выполнять расчеты напряжений и перемещений методом конечных элементов
- Моделировать дефекты и оценивать их влияние
- Использовать нейросеть для предсказания местоположения дефектов
- Генерировать отчеты в PDF
- Обучать нейросеть на новых данных

## Три сервиса

### 1. Основной фронтенд (Конструктор)

Визуальный 2D редактор стержневых систем с:

- Canvas на Fabric.js
- Формами редактирования параметров
- Визуализацией результатов расчетов
- Отображением предсказаний нейросети
- Генерацией PDF отчетов

### 2. Бэкенд

FastAPI сервер с:

- МКЭ решателем (NumPy/SciPy)
- REST API для всех операций
- Интеграцией нейросети из Дип.ipynb
- Celery для асинхронного обучения
- PostgreSQL и Redis

### 3. Дашборд обучения

Отдельный фронтенд для:

- Просмотра и редактирования датасета
- Мониторинга обучения нейросети
- Управления версиями моделей

## Ключевые особенности

- Визуальный конструктор (НЕ через формы, формы только для редактирования)
- МКЭ расчеты на бэкенде
- Дефекты с концентрацией напряжений
- Нейросетевое предсказание дефектов
- Тепловая карта вероятности
- PDF отчеты с эпюрами и схемами
- Real-time мониторинг обучения
- Запуск одной командой docker-compose up
